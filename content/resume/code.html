I joined the Center for Open Data Enterprise as a sole developer to work on the UN SDG National Reporting Initiative project, a platform aimed at helping national organizations share knowledge on implementing reporting for the UN Sustainable Development Goals. After the initial UN SDG National Reporting Initiative project stage ended, I continued to assist with additional projects and volunteered maintenance time.

Highlights

  • UN SDG National Reporting Initiative website:
    • Helped drive the design process in cooperation with Threespot, creating multiple quick website prototypes throughout the process to guide decision-making.
    • Implemented and deployed the website utilizing a Jamstack solution (Jekyll, Vue.js + Vuex, Netlify, and PostgreSQL-backed Express API servers utilizing Sequelize) combined with services such as Forest Admin for backend data entry to better leverage limited financial and technical capacity.
    • Implemented filtered keyword search strictly through PostgreSQL to minimize resource usage and maintenance costs from infrastructure complexity.
  • Overhauled CODE's Open Data Impact Map website (JavaScript and PHP), including relational schema redesigns, bugfixing, code refactoring (eliminated ~20k lines of code), and infrastructure updates.
  • Implemented a redesign/relaunch of the main CODE website (WebFlow).
  • Volunteered ongoing time to help maintain the CODE website.
