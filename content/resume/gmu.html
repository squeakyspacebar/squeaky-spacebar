I assisted a colleague with bringing an unoptimized Leaflet map visualization for the Urban Renewal Impact website into a performantly working state for a project deadline.

Highlights

  • Migrated the front end from jQuery to Vue.js + Vuex to better manage the application's complexity, improving readability, extensibility, and maintainability.
  • Rewrote existing API endpoints (Slim PHP framework) and optimized SQL queries to fetch data in a more targeted manner while migrating heavy processing and memory demands away from the front end by leveraging the backing PostgreSQL database.
  • Eliminated large portions of inefficient and redundant code.
